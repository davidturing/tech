# 《Python数据分析》课程大纲 (2026年春季学期)

## 课程基本信息
- **课程名称**: Python数据分析
- **适用对象**: 985高校工科本科生/研究生
- **学分**: 3学分
- **学时**: 48学时（理论32 + 实验16）
- **先修课程**: Python程序设计基础、高等数学、概率论与数理统计

## 课程目标
本课程旨在培养学生掌握现代Python数据分析的核心技能，结合2026年最新技术趋势，使学生能够：
1. 熟练使用Python进行数据获取、清洗、分析和可视化
2. 掌握主流数据分析库（pandas, numpy, matplotlib, seaborn等）的高级用法
3. 了解AI辅助数据分析的最新发展（如大模型在数据分析中的应用）
4. 具备解决实际工程问题的数据分析能力
5. 培养数据驱动的思维方式和工程实践能力

## 课程特色
- **现代化**: 融入2026年最新技术（AI辅助编程、自动化数据分析等）
- **实践导向**: 每章配套真实案例和动手实验
- **渐进式**: 从基础到高级，循序渐进
- **工程化**: 强调代码质量和工程最佳实践
- **可视化**: 图文并茂，直观易懂

## 详细教学安排

### 第一部分：基础篇（第1-4周）

**第1周：Python数据分析环境搭建与基础**
- JupyterLab vs VSCode + Jupyter扩展
- Conda/Mamba环境管理（2026年推荐使用Mamba）
- Git版本控制与GitHub协作
- Python基础回顾（列表推导式、生成器、装饰器等）
- 实验1：环境配置与第一个数据分析项目

**第2周：NumPy数组计算基础**
- ndarray核心概念与内存布局
- 向量化操作与广播机制
- 高级索引与切片技巧
- 性能优化：避免Python循环
- 实验2：图像处理中的NumPy应用

**第3周：Pandas数据结构与基础操作**
- Series与DataFrame核心概念
- 数据读取与写入（CSV, Excel, JSON, Parquet）
- 基础数据清洗：缺失值、重复值处理
- 数据类型转换与内存优化
- 实验3：股票数据初步分析

**第4周：Pandas数据操作进阶**
- 分组聚合操作（groupby）
- 数据透视表（pivot_table）
- 合并与连接（merge, join, concat）
- 时间序列基础处理
- 实验4：电商用户行为分析

### 第二部分：进阶篇（第5-8周）

**第5周：数据可视化基础**
- Matplotlib核心概念与面向对象API
- Seaborn统计可视化
- Plotly交互式可视化（2026年重点）
- 可视化最佳实践与配色方案
- 实验5：多维度数据可视化项目

**第6周：高级数据清洗与特征工程**
- 正则表达式在数据清洗中的应用
- 文本数据处理（字符串方法、正则）
- 特征编码：独热编码、标签编码、目标编码
- 特征缩放与标准化
- 实验6：文本数据清洗与特征提取

**第7周：时间序列分析**
- Pandas时间序列功能深度使用
- 移动窗口计算
- 时间序列分解与趋势分析
- 季节性调整与预测基础
- 实验7：股票价格时间序列分析

**第8周：统计分析与假设检验**
- 描述性统计与分布分析
- 相关性分析（Pearson, Spearman）
- 假设检验基础（t检验、卡方检验）
- 置信区间与效应量
- 实验8：A/B测试数据分析

### 第三部分：现代篇（第9-12周）

**第9周：机器学习基础与Scikit-learn**
- 机器学习基本概念与流程
- Scikit-learn API设计哲学
- 模型训练、验证与评估
- 交叉验证与超参数调优
- 实验9：房价预测项目

**第10周：AI辅助数据分析（2026年新内容）**
- 大语言模型在数据分析中的应用
- AI代码生成与自动文档
- 自然语言到代码转换（NL2Code）
- 智能数据探索与洞察发现
- 实验10：使用AI助手完成数据分析任务

**第11周：大数据处理基础**
- Dask并行计算框架
- 内存不足情况下的数据处理策略
- 数据分块处理与流式处理
- 性能监控与优化
- 实验11：大型数据集处理实战

**第12周：综合项目与最佳实践**
- 完整数据分析项目流程
- 代码质量与可维护性
- 文档编写与结果展示
- 团队协作与版本控制最佳实践
- 期末项目：自选领域数据分析

## 考核方式
- **平时作业**: 40%（每周实验 + 编程作业）
- **期中项目**: 20%（第8周，时间序列分析项目）
- **期末项目**: 40%（完整数据分析项目，包含代码、报告、演示）

- **开发环境**: VSCode + Jupyter扩展 或 JupyterLab 4.0+
- **包管理**: conda/mamba
- **版本控制**: Git + GitHub
- **AI工具**: GitHub Copilot, CodeWhisperer 等AI编程助手
