# 《Python数据分析》课程 - 2026年现代化版本

## 课程概述
本课程专为中国985高校工科学生设计，结合2026年最新技术趋势，从基础到高级系统性地教授Python数据分析技能。课程强调理论与实践相结合，培养学生的数据思维和实际编程能力。

## 课程特色
- 🎯 **循序渐进**：从Python基础到高级数据分析技术
- 🛠️ **动手实践**：每章配套Jupyter Notebook练习
- 📊 **现代工具**：涵盖2026年主流数据分析工具链
- 🎨 **图文并茂**：丰富的可视化示例和图表
- 💡 **案例驱动**：真实世界的数据分析案例

## 课程结构
```
course/
```

## 技术栈 (2026)
- **核心语言**: Python 3.12+
- **数据处理**: pandas 2.0+, polars, vaex
- **可视化**: matplotlib 3.8+, seaborn 0.13+, plotly 5.20+, altair 5.0+
- **机器学习**: scikit-learn 1.5+, xgboost 2.0+, lightgbm 4.0+
- **深度学习**: PyTorch 2.5+, TensorFlow 2.15+
- **大模型集成**: LangChain, LlamaIndex
- **开发环境**: JupyterLab 4.0+, VS Code + Copilot
- **部署工具**: Streamlit 1.35+, Gradio 4.0+

## 适用对象
- 985高校工科本科生（大二及以上）
- 具备基础Python编程知识
- 对数据分析和人工智能感兴趣的学生
