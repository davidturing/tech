# 项目一：数据清洗与预处理实战

## 项目目标
- 掌握真实世界数据的清洗技巧
- 学会处理缺失值、异常值和重复数据
- 理解数据质量对分析结果的影响

## 项目背景
在2026年，数据已经成为最宝贵的资源之一。然而，现实世界的数据往往存在各种问题：缺失值、格式不一致、异常值等。本项目将使用来自某电商平台的真实用户行为数据，让学生体验完整的数据清洗流程。

## 技术栈
- **Python 3.12+**：最新版本的Python语言
- **Pandas 2.2+**：数据处理的核心库
- **Polars**：高性能替代方案（2026年新兴技术）
- **Great Expectations**：数据质量验证框架

## 项目任务
### 基础任务（必做）
1. **数据加载与初步探索**
   - 使用 `pd.read_csv()` 加载数据
   - 查看数据基本信息（shape, info, describe）
   - 识别数据类型和潜在问题

2. **缺失值处理**
   - 统计各列缺失值比例
   - 根据业务逻辑选择合适的填充策略
   - 实现自动化缺失值处理函数

3. **异常值检测与处理**
   - 使用 IQR 方法检测数值型异常值
   - 使用正则表达式清理文本型异常值
   - 实现异常值可视化

### 进阶任务（选做）
1. **性能优化**
   - 对比 Pandas 和 Polars 的处理速度
   - 实现并行数据处理
   - 内存优化技巧

2. **数据质量报告**
   - 使用 Great Expectations 生成数据质量报告
   - 自动化数据质量监控
   - 构建数据质量仪表板

## 评估标准
