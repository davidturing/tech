# 项目二：机器学习预测管道

## 项目目标
构建一个完整的机器学习预测管道，从数据预处理到模型部署。

## 技术栈
- **数据处理**: Pandas, Scikit-learn
- **模型训练**: Scikit-learn, XGBoost
- **模型评估**: Yellowbrick, SHAP
- **管道构建**: Scikit-learn Pipeline

## 数据集
使用 Kaggle 上的房价预测数据集（2026年更新版本）

## 项目步骤

### 第1周：数据探索与特征工程
- 加载和理解数据
- 处理缺失值和异常值
- 特征编码和转换
- 特征选择

### 第2周：模型训练与验证
- 划分训练/验证/测试集
- 基线模型建立
- 超参数调优
- 交叉验证

### 第3周：模型解释与优化
- 使用 SHAP 进行模型解释
- 特征重要性分析
- 模型集成
- 性能优化

### 第4周：管道部署
- 构建完整的预测管道
- 模型序列化
- 创建简单的 Web API
- 编写使用文档

## 交付物
- Jupyter Notebook 完整代码
- 模型性能报告
- 特征重要性可视化
- 部署说明文档

## 评估标准
- 代码质量 (30%)
- 模型性能 (25%)
- 分析深度 (25%)
- 文档完整性 (20%)

## 扩展挑战
- 实现自动化的超参数调优
- 添加模型监控功能
- 集成 LLM 辅助特征工程
