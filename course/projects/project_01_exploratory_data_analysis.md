# 项目1: 探索性数据分析 (EDA) - 全球AI发展趋势分析

## 项目概述
本项目将引导学生使用Python进行完整的探索性数据分析流程，分析2026年全球AI发展趋势数据集。学生将学习如何从原始数据中提取洞察，并创建专业的可视化报告。

## 学习目标
- 掌握Pandas数据加载和基本操作
- 理解数据质量评估方法
- 学会使用Seaborn和Plotly创建专业可视化
- 培养数据故事叙述能力

## 数据集
- **来源**: Kaggle 2026 AI Trends Dataset
- **包含字段**:
  - `country`: 国家/地区
  - `year`: 年份 (2020-2026)
  - `ai_investment_millions`: AI投资金额(百万美元)
  - `ai_patents`: AI相关专利数量
  - `ai_talent_count`: AI人才数量
  - `ai_companies`: AI公司数量
  - `government_support_score`: 政府支持度评分(1-10)

## 任务要求

### 基础任务 (必做)
1. **数据加载与检查**
   - 使用`pd.read_csv()`加载数据
   - 检查数据基本信息 (`info()`, `describe()`)
   - 识别缺失值和异常值

2. **数据清洗**
   - 处理缺失值（删除或填充）
   - 标准化国家名称
   - 验证数据一致性

3. **基础分析**
   - 计算各国AI投资的年增长率
   - 找出AI人才密度最高的前10个国家
   - 分析政府支持度与AI投资的相关性

4. **可视化**
   - 创建时间序列图展示全球AI投资趋势
   - 制作热力图显示各国AI发展指标相关性
   - 使用散点图矩阵展示多变量关系

### 进阶任务 (选做)
1. **地理可视化**
   - 使用Plotly创建交互式世界地图
   - 添加悬停信息显示详细指标

2. **预测分析**
   - 使用线性回归预测2027年AI投资趋势
   - 评估模型性能

3. **报告撰写**
   - 使用Jupyter Notebook创建完整的分析报告
   - 包含执行摘要、方法论、结果和建议

## 交付物
- Jupyter Notebook 文件 (.ipynb)
- 清洗后的数据集 (.csv)
- 可视化图片文件夹
- 项目报告 (Markdown格式)

- **可视化效果** (25%): 图表专业、信息传达清晰
