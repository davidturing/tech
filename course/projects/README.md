# Python数据分析实践项目

本课程包含4个递进式实践项目，每个项目都基于真实世界的数据集和应用场景。

## 项目设计理念
- **真实性**: 使用2026年最新的真实数据集
- **实用性**: 解决实际业务问题
- **渐进性**: 从简单到复杂，逐步提升技能
- **完整性**: 包含数据获取、清洗、分析、可视化、报告全流程

## 项目列表

### 📊 项目1: 数据清洗与基础分析 (第3周)
**主题**: 全球AI发展趋势数据分析  
**技能点**: Pandas基础操作、数据清洗、描述性统计  
**交付物**: Jupyter Notebook + 分析报告

### 📈 项目2: 可视化与探索性分析 (第6周)  
**主题**: 中国碳中和进展数据可视化
**技能点**: Matplotlib/Seaborn/Plotly、交互式图表、故事化数据呈现
**交付物**: 交互式仪表板 + 可视化报告

### 🔍 项目3: 统计建模与预测 (第9周)
**主题**: 股票价格预测与风险分析
**技能点**: 时间序列分析、回归模型、假设检验
**交付物**: 预测模型 + 风险评估报告

### 🤖 项目4: 端到端AI数据分析系统 (第12周)
**主题**: 构建智能数据分析助手
**技能点**: API集成、自动化分析、LLM辅助决策
**交付物**: 完整的Web应用 + 技术文档

---
*每个项目都提供详细的需求说明、数据集、评估标准和参考解决方案*
