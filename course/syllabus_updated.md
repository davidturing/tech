# 《Python数据分析》课程大纲（2026版）

## 课程目标
本课程旨在培养学生掌握现代Python数据分析的完整技能栈，从基础数据处理到高级AI集成应用，具备解决实际问题的能力。

## 核心技术栈
- **数据处理**: Pandas 3.0, Polars, Dask
- **可视化**: Plotly, Seaborn, Matplotlib, Altair  
- **统计分析**: SciPy, Statsmodels, Pingouin
- **机器学习**: Scikit-learn, XGBoost, LightGBM
- **AI集成**: LLM辅助分析, AutoML
- **开发环境**: JupyterLab, VS Code, GitHub Codespaces

## 详细周计划

### 第1-2周：Python数据分析基础
- Python基础回顾与最佳实践
- Jupyter Notebook/Lab 高效使用
- NumPy 2.0 数组操作与性能优化
- Pandas 3.0 核心概念：Series, DataFrame, Index
- 数据导入导出：CSV, JSON, Excel, Parquet, Feather

### 第3-4周：数据清洗与预处理
- 缺失值处理策略
- 异常值检测与处理
- 数据类型转换与优化
- 字符串操作与正则表达式
- 时间序列数据处理
- Polars vs Pandas 性能对比

### 第5-6周：数据探索与可视化
- 描述性统计分析
- Plotly交互式可视化
- Seaborn统计图表
- Matplotlib定制化图表
- 地理空间数据可视化
- 仪表板构建 (Dash/Streamlit)

### 第7-8周：统计推断与假设检验
- 概率分布与抽样
- 置信区间与假设检验
- 方差分析 (ANOVA)
- 相关性与回归分析
- 非参数检验方法
- 多重比较校正

### 第9-10周：机器学习基础
- 监督学习 vs 无监督学习
- 特征工程与选择
- 模型评估与验证
- 回归与分类算法
- 聚类分析
- AutoML工具使用

### 第11-12周：高级主题与项目实战
- 大数据处理：Dask, Vaex
- Web API数据获取
- 实时数据流处理
- LLM辅助数据分析
- 完整端到端项目
- 代码部署与分享

## 实践项目参考（基于freeCodeCamp模式）

### 项目1：数据清洗大师
- **目标**: 掌握各种数据质量问题的处理方法
- **数据集**: 真实世界脏数据集（包含缺失值、异常值、格式不一致等）
- **产出**: 清洗报告 + 可复用的清洗函数库

### 项目2：探索性数据分析
- **目标**: 全面探索数据特征，发现洞察
- **数据集**: 2026年AI发展趋势数据 / 碳中和相关数据
- **产出**: 交互式仪表板 + 分析报告

### 项目3：预测建模
- **目标**: 构建准确的预测模型
- **数据集**: 时间序列数据（如股票价格、天气数据）
- **产出**: 预测模型 + 性能评估报告

### 项目4：端到端数据分析应用
- **目标**: 完整的数据分析工作流
- **数据集**: 自选真实数据集
- **产出**: Web应用 + 技术文档 + 演示视频

## 评估方式
