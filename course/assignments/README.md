# 《Python数据分析》课程作业体系

## 作业设计理念
- **循序渐进**：从基础操作到综合项目
- **实战导向**：基于真实数据集和实际问题
- **能力培养**：兼顾编程技能、数据思维和解决问题能力
- **2026年技术前沿**：融入最新的数据分析工具和方法

## 作业结构

### 基础练习 (Weekly Exercises)
- 每周1-2次基础练习
- 重点巩固当周知识点
- 自动化评分 + 人工反馈

### 项目作业 (Project Assignments)
- 每2-3周一个中型项目
- 综合运用多个知识点
- 团队协作 + 个人贡献

### 期末大项目 (Final Project)
- 学期末综合性项目
- 自选主题 + 导师指导
- 完整的数据分析流程

## 技术栈要求 (2026年标准)
- **核心库**: pandas 2.0+, numpy 2.0+, matplotlib 4.0+, seaborn 0.13+
- **新兴工具**: polars (高性能替代), duckdb (嵌入式分析)
- **可视化**: plotly 6.0+, altair 5.0+, pyecharts (中文支持)
- **机器学习**: scikit-learn 1.5+, xgboost 2.0+
- **深度学习**: tensorflow 2.15+, pytorch 2.5+ (可选)
- **部署工具**: streamlit 1.35+, gradio 4.0+

## 评分标准
- **代码质量** (30%): 可读性、效率、文档
- **分析深度** (30%): 问题理解、方法选择、结果解释  
- **创新性** (20%): 解决方案的独特性、技术前沿性
- **呈现效果** (20%): 可视化质量、报告结构、演示能力

## 提交格式
- GitHub 仓库 (包含完整代码和文档)
- Jupyter Notebook (交互式分析过程)
- PDF 报告 (正式分析结果)
- 演示视频 (5-10分钟项目展示)
