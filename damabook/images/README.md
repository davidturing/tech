# DAMA 蓝图图像生成标准

## 生成模板
1. **标准**：浅色背景、高对比度文字、16:9比例、2K分辨率，PNG格式
2. **命名**：PNG图片的正确文件命名 (chapter_XX.png)
3. **存储**：存放到damabook的子目录 images目录下
4. **发布**：https://github.com/davidturing/tech/blob/main/damabook/images/
5. **API**：使用G老师的生成图片API，遇到503限流就持续定时重试，不切换其他方法
