# 仿生双脑多智能体架构演进报告（2026 最新版）

## 一、当前架构总览
- 左脑（逻辑脑）：工程架构、代码实现、模型调度、数据治理
- 右脑（创造脑）：内容生成、多模态创作、表达输出
- 虚拟胼胝体：双脑通信、状态协同、任务调度
- 海马体（Memory Alpha）：三轨记忆、认知熵减、长期知识存储
- 模型网关：多模型自动注册、智能路由、故障转移、高可用
- 数字分身矩阵：数据专家、教育老师、科技博主、创作者等

## 二、已落地核心能力
- 海马体知识库：本地 + iCloud 安全同步（仅文档，不上传密钥/配置）
- 每15分钟自动同步：rsync + launchd 稳定运行
- 智能文件过滤：只同步 md / png / jpg / pdf
- 多模型统一网关：Gemini / GLM / 豆包 / Qwen 自动路由
- 数字分身协同：按角色自动分配任务
- 可审计架构：所有变更可回溯、可复盘

## 三、技术基座
- 语言：Python
- 记忆存储：LanceDB + SQLite
- 同步：iCloud Drive
- 笔记：Obsidian
- 自动化：定时同步、文档发布、版本管理
